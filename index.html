Predicción de Mora en Créditos Bancarios

Autor: Edwin Céspedes Alvarado

📁 Ver Repositorio en GitHub ▶️ Ver Explicación en YouTube
Este proyecto utiliza Redes Neuronales Profundas para identificar patrones de riesgo crediticio. Se implementaron técnicas avanzadas de preprocesamiento y arquitectura de modelos con TensorFlow/Keras.
Modelo:
Red Neuronal Profunda (DNN)
Técnica Clave:
Balanceo de datos con SMOTE

Contenido del Proyecto:

  • Análisis Exploratorio de Datos (EDA) interactivo.
  • Tratamiento de valores atípicos y escalado de variables.
  • Optimización de hiperparámetros.
  • Evaluación con matrices de confusión y curvas ROC/AUC.
