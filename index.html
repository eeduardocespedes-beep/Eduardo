Predicción de Mora en Créditos Bancarios

Por: Edwin Céspedes Alvarado

📁 Repositorio GitHub ▶️ Video Explicativo 📄 Presentación PDF
Implementación de una Red Neuronal Profunda (DNN) para la detección temprana de riesgos crediticios. El proyecto aborda el desbalanceo de datos y busca maximizar la interpretabilidad de los modelos de inteligencia artificial.
Modelo Principal
Red Neuronal Profunda
Optimización
SMOTE & Dropout layers

Puntos Clave del Proyecto:

  • Limpieza y preprocesamiento de datos bancarios masivos.
  • Entrenamiento de modelos supervisados con TensorFlow.
  • Análisis de métricas: Precisión, Recall y Curva ROC.
  • Interpretación de resultados con librerías especializadas.
